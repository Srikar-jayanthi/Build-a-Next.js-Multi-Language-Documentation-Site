# Einführung

Willkommen auf der **Dokumentationsplattform**. Dieses Projekt demonstriert die Fähigkeiten von Next.js einschließlich:
- ISR
- i18n
- Tailwind CSS

## Erste Schritte

Um zu beginnen, führen Sie einfach den Docker-Container aus.

```bash
docker-compose up
```
