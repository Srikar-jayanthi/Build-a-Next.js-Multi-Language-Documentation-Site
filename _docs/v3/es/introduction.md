# Introducción

Bienvenido a la **Plataforma de Documentación**. Este proyecto demuestra las capacidades de Next.js incluyendo:
- ISR
- i18n
- Tailwind CSS

## Empezando

Para empezar, simplemente ejecuta el contenedor docker.

```bash
docker-compose up
```
