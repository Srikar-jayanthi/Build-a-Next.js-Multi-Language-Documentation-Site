# Introduction

Bienvenue sur la **Plateforme de Documentation**. Ce projet démontre les capacités de Next.js incluant:
- ISR
- i18n
- Tailwind CSS

## Commencer

Pour commencer, exécutez simplement le conteneur docker.

```bash
docker-compose up
```
